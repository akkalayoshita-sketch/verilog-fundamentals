# Verilog Fundamentals

A collection of fundamental combinational circuit designs implemented using **Verilog HDL**. This repository is part of my learning journey in **Digital Design** and **VLSI Design**.

## 📂 Circuits Included

- Full Adder
- 4:2 Encoder 
- 2:4 Decoder
- Logic gates
- Full Adder by using Hlaf Adder 
- 2:1 Multiplexer (MUX)
- 4:1 Multiplexer by using 2:1 Multiplier (MUX)
- Comparator

Each circuit includes:
- Verilog HDL design (`.v`)
- Testbench (`_tb.v`) for functional verification

## 🛠️ Tools Used

- Verilog HDL
- Synopsys Design Compiler (DC)
- Synopsys VCS 
- Linux Environment

## 📚 Topics Covered

- Combinational Logic Design
- RTL Coding
- Functional Verification using Testbenches
